# Employee Attrition Prediction using Machine Learning

## Overview

This project analyzes employee attrition using the IBM HR Analytics Employee Attrition dataset. The objective is to identify the factors that contribute to employee turnover and provide actionable insights that can help HR teams improve employee retention.

The project follows a complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), model training, evaluation, visualization, and business recommendations.

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)

## Key Business Insights

- Employee attrition is influenced by multiple workplace factors rather than salary alone.
- Job satisfaction, work-life balance, overtime, and employee tenure are strong indicators of employee turnover.
- Attrition is higher among employees in their early years with the company.
- The analysis highlights departments and job roles that require focused retention strategies.

---

## HR Recommendations

- Improve onboarding and mentoring programs for new employees.
- Monitor employee workload and overtime.
- Promote better work-life balance.
- Conduct regular employee engagement and feedback sessions.
- Focus retention efforts on departments and roles with higher attrition.
